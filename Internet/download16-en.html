For newer versions of OpenXP/16 see OpenXP16.de.

Updates consist of online help files and executables. Complete versions include documentation also.
Complete versions are compressed using zip (see WinZip.com), all other packages are compressed with rar (unrar.exe, 90kb).
Caution when decompressing archives: Subdirectories are included, so use parameter -d (zip) or -r (rar) to decompress.

Some other things that may be of interest can be found on the OpenXP ftp server.
