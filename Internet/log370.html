  • JG:- Fix: When browsing through messages with <Ctrl-PgUp>/<Ctrl-PgDn> and +/- the message that has been entered with the last <Ctrl-Pg*> was replaced in the message window with the message the message reader had entered first
  • JG: Fix (of an extremely ancient and annoying behaviour): When creating an (e.g. Reply-To) user upon replying to a message stored in a user folder, XP defaults to the server of the user folder where the message is stored rather than to the primary server. Finally!

  • MA: SMTP-Envelope ist fix Edit/Boxen/Edit/Namen, damit und mit dem Role-Feature lassen sich jetzt auch bei Providern frei die Absenderadresse ändern, die das eigentlich verhindern (z.B. GMX)

  • MA: GnuPG-Support
