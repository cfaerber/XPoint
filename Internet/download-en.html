Please check the following points before posting OpenXP/32 bug reports:

- check bug list and OpenXP-Dev archive
...and, finally, please include debuglog.txt whenever you submit bug reports.

Updates consist of online help files and executables. Complete versions include documentation also.
Complete versions are compressed using zip (see WinZip.com), all other packages are compressed with rar (unrar.exe, 32kb).
Caution when decompressing archives: Subdirectories are included, so use parameter -d (zip) or -r (rar) to decompress.
